# GitHub Pages Launch

This project is ready for GitHub Pages.

## What to upload

Upload the whole project, including:

- `index.html`
- `styles.css`
- `src/`
- `assets/`
- `.github/workflows/deploy-pages.yml`
- `.nojekyll`

## Steps

1. Create a new GitHub repository.
2. Upload all project files to the `main` branch.
3. Open the repo on GitHub.
4. Go to `Settings` -> `Pages`.
5. Under `Build and deployment`, choose `GitHub Actions`.
6. Push any change or open the `Actions` tab and run `Deploy Romantic Site`.
7. After deployment, GitHub will give you a `github.io` website URL.

## Custom domain

1. In `Settings` -> `Pages`, add your domain.
2. Update your domain DNS with the records GitHub shows you.
3. Wait for GitHub Pages SSL to finish provisioning.

## Important

The site is static, so GitHub Pages is enough. No server is needed.
